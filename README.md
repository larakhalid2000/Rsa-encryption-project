# RSA Encryption & Decryption (Java)

## Project Overview
This project presents an academic implementation and explanation of the RSA public-key cryptography algorithm.
The project focuses on key generation, encryption, and decryption concepts using Java.

## Project Details
- Explanation of RSA algorithm steps
- Java-based implementation (included as code screenshots in the report)
- Use of BigInteger and modular arithmetic
- Encryption and decryption process walkthrough
